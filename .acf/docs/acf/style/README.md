# Communication Styles

## Style Selection Matrix

| Context | Style Guide | Who Uses It |
|---------|-------------|-------------|
| **General communication** | general.md | All agents in normal interactions |
| **Task descriptions** | task-descriptions.md | task-engineer when creating tasks |
| **Research documents** | research-documents.md | Any agent writing research docs |

## Purpose

Different contexts require different communication patterns:

- **General**: Technical discussions, code reviews, specifications (all agents)
- **Task**: Binary outcomes, concrete measurements (task-engineer only)
- **Research**: Evidence-based analysis, sourced claims (research work)

See individual style guides for detailed rules.
